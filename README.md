# Divine Fantasy: Whispers

A web-based RPG that blends social simulation, skilling, and strategic conquest in a dark fantasy world.

## Project Structure

- `GDD/` - Game Design Documents and planning
- `divine_fantasy/` - The React game application

## Getting Started

1. Navigate to the game directory:
   ```bash
   cd divine_fantasy
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to `http://localhost:3000`
