# Game Design Document: "Divine Fantasy: Whispers"

## 1. Core Concept & Vision

**Project Name:** Divine Fantasy: Whispers

**Core Concept:** A UI-based, single-player RPG that strategically blends core mechanics from three distinct genres:
- **Sims-Style Social Simulation:** Deep NPC interaction, relationship management, and personal stat development.
- **RuneScape-Style Skilling:** A comprehensive system of non-combat and combat skills (e.g., woodcutting, blacksmithing, etc.) that form a core progression loop.
- **Mount & Blade-Style Strategy:** A high-level strategic layer involving army management, companion recruitment, and territory control.

**Platform:** Web (React)

**UI Aesthetic:** A dark, modern, and clean user interface, utilizing high-quality location imagery and subtle gradient effects. This custom aesthetic will be built using **Tailwind CSS** to ensure maximum design flexibility.

## 2. Target Audience

This game is for players who enjoy deep, complex RPGs with a focus on character progression and world-building. The target audience includes fans of games like *The Sims*, *RuneScape*, and *Mount & Blade*, who are looking for a new experience that combines the best elements of these genres.

## 3. Project Scope (v1.0)

The initial release (v1.0) will serve as a playable introduction to the world and core mechanics. The main features for v1.0 will be:
- The initial implementation of the "Skilling" and "Relationship" loops.
- A preview of the "Conquest" loop.
- The introductory chapter of the story for the starting character, "Luke".
- A portion of the Driftwatch area, featuring key NPCs and introductory quests.

## 4. Key Features

- **Genre-Bending Gameplay:** A unique blend of social simulation, skilling, and strategy.
- **Deep Skill System:** A vast array of skills to master, with an exponential XP curve for a rewarding progression.
- **Complex NPC Relationships:** Go beyond simple dialogue with a multi-faceted relationship system.
- **Strategic Conquest:** Rise from a humble orphan to a powerful lord, commanding armies and conquering territories.
- **Immersive World:** A rich world with a deep history, intriguing lore, and memorable characters.
